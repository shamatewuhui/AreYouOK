## [1.0.0] - 2026-02-04

### 新增 (Added)

#### 核心功能
- ✨ 用户签到功能
  - 支持姓名和邮箱输入
  - 表单验证（邮箱格式、姓名长度）
  - 签到成功提示和震动反馈
  - 防止重复签到

- ✨ 用户信息页面 (`pages/user/info.vue`)
  - 用户头像、昵称、邮箱展示
  - 邮箱验证状态标识
  - 签到统计：连续签到天数、累计签到天数
  - 最后签到时间智能显示
  - 刷新用户信息功能
  - 快捷跳转到签到记录

- ✨ 签到记录页面 (`pages/sign/records.vue`)
  - 签到记录列表展示
  - 日期和时间格式化显示
  - 分页加载功能（每页20条）
  - 上拉加载更多
  - 下拉刷新
  - 日期筛选功能（开始日期、结束日期）
  - 空状态友好提示

#### 工具和配置
- 📦 全局状态管理 (`utils/store.js`)
  - 用户信息状态管理
  - 登录状态管理
  - 数据持久化到本地存储
  - 签到信息更新方法

- 🔧 配置文件 (`utils/config.js`)
  - 开发环境和生产环境配置
  - API基础地址配置
  - 应用信息配置
  - 本地存储键名管理

- 🌐 API接口工具优化 (`utils/api.js`)
  - 统一请求封装
  - 支持完整的查询参数
  - 错误处理优化
  - 新增通过邮箱查询用户信息接口

#### UI/UX优化
- 🎨 首页优化
  - 用户信息快捷入口
  - 签到统计展示
  - 底部功能按钮
  - 页面状态同步

- 🎨 设计风格
  - 温暖精致的配色方案
  - 流畅的动画效果
  - 背景装饰元素
  - 圆形签到按钮动画
  - 卡片式布局

- 📝 前端开发完善说明 (`前端开发完善说明.md`)
- 📝 部署配置说明 (`部署配置说明.md`)
- 📝 静态资源说明 (`static/README.md`)

### 优化 (Changed)

- ⚡ 优化API请求工具类
  - 支持完整的分页参数
  - 支持日期范围筛选
  - 增强错误提示

- 💄 优化页面路由配置
  - 统一导航栏背景色
  - 签到记录页面启用下拉刷新
  - 添加页面调试快捷入口

- 🔄 优化数据流转
  - 签到成功后自动更新用户信息
  - 页面显示时自动刷新状态
  - 本地数据持久化

### 技术栈 (Tech Stack)

- **框架**: uni-app 3.x + Vue 3
- **构建工具**: Vite 5.x
- **样式**: SCSS
- **状态管理**: 自定义 Store（基于 uni.storage）
- **UI组件**: 自定义组件
