# 本地编译说明

## 编译方法

项目已配置好本地编译脚本，无需安装 npm 依赖即可编译。

### 编译步骤

1. **运行编译脚本**：
   ```bash
   cd AreYouOK-uniapp
   node build.js
   ```

2. **编译完成后**，会在 `dist/dev/mp-weixin` 目录生成微信小程序代码。

3. **在微信开发者工具中打开**：
   - 打开微信开发者工具
   - 选择"导入项目"
   - **项目目录**：选择 `AreYouOK-uniapp/dist/dev/mp-weixin`
   - **AppID**：`wx9d3afb740a541e4e`
   - 点击"导入"

## 注意事项

⚠️ **重要**：由于 Vue 到微信小程序的自动转换比较复杂，生成的文件可能需要手动调整：

1. **WXML 文件**：可能包含 Vue 指令（如 `v-model`, `v-if`），需要手动转换为小程序语法
2. **JS 文件**：Vue 的 `data()`, `computed`, `methods` 需要手动转换为小程序 Page 格式
3. **API 调用**：`uni.xxx` 已自动转换为 `wx.xxx`，但可能需要检查

## 如果编译后的代码有问题

### 方案 1：手动修复生成的文件

在 `dist/dev/mp-weixin` 目录中手动修复：
- `pages/index/index.js` - 转换为小程序 Page 格式
- `pages/index/index.wxml` - 转换 Vue 指令为小程序语法

### 方案 2：使用 HBuilderX（推荐）

如果本地编译遇到问题，建议使用 HBuilderX：
1. 下载安装 HBuilderX
2. 在 HBuilderX 中打开项目
3. 运行 → 运行到小程序模拟器 → 微信开发者工具

## 重新编译

每次修改源码后，需要重新运行编译脚本：
```bash
node build.js
```

然后刷新微信开发者工具中的项目。

## 项目结构

```
AreYouOK-uniapp/
├── pages/              # 源码页面
├── utils/              # 工具函数
├── App.vue            # 应用入口
├── pages.json         # 页面配置
├── manifest.json      # 应用配置
├── build.js           # 编译脚本
└── dist/              # 编译输出目录
    └── dev/
        └── mp-weixin/ # 微信小程序编译结果（在微信开发者工具中打开此目录）
```
