# 上线否 - 微信小程序

一个简单的每日签到小程序，用户可以输入姓名和联系人邮箱，点击"上线"按钮进行每日签到。

## 项目结构

```
AreYouOK-uniapp/
├── pages/              # 页面目录
│   └── index/         # 首页
│       └── index.vue  # 主页面
├── utils/             # 工具函数目录
│   ├── api.js        # API 请求工具
│   └── validate.js  # 表单验证工具
├── App.vue           # 应用入口文件
├── pages.json        # 页面配置
├── manifest.json     # 应用配置
└── README.md        # 项目说明文档
```

## 功能特性

- ✅ 用户输入姓名和联系人邮箱
- ✅ 表单验证（邮箱格式、姓名格式）
- ✅ 每日签到功能
- ✅ 本地存储用户信息（自动保存）
- ✅ 签到状态提示
- ✅ 响应式设计，适配不同屏幕尺寸
- ✅ 优雅的 UI 设计

## 技术栈

- **框架**: uni-app (Vue 3)
- **平台**: 微信小程序
- **样式**: SCSS

## 开发说明

### 1. 配置后端 API 地址

在 `utils/api.js` 文件中修改 `BASE_URL` 为实际的后端 API 地址：

```javascript
const BASE_URL = 'https://your-api-domain.com/api'
```

### 2. 后端 API 接口要求

#### 签到接口

**接口地址**: `POST /api/sign`

**请求参数**:
```json
{
  "name": "用户姓名",
  "email": "联系人邮箱",
  "openid": "微信openid（可选）"
}
```

**响应格式**:
```json
{
  "code": 0,
  "message": "签到成功",
  "data": {
    "signDate": "2026-02-03",
    "continuousDays": 5,
    "totalSignDays": 30
  }
}
```

### 3. 运行项目

1. 使用 HBuilderX 打开项目
2. 配置微信小程序 AppID（在 `manifest.json` 中）
3. 点击运行 -> 运行到小程序模拟器 -> 微信开发者工具

### 4. 构建发布

1. 点击发行 -> 小程序-微信
2. 填写小程序相关信息
3. 上传代码到微信公众平台

## 页面说明

### 主页面 (pages/index/index.vue)

- **姓名输入框**: 用户输入自己的姓名（2-20个字符）
- **邮箱输入框**: 用户输入联系人邮箱（用于接收提醒邮件）
- **签到按钮**: 居中的大尺寸圆形按钮，点击后发送签到请求

## 数据存储

应用会在本地存储用户输入的姓名和邮箱，下次打开时自动填充，提升用户体验。

## 注意事项

1. 需要在 `manifest.json` 中配置微信小程序的 AppID
2. 需要在 `utils/api.js` 中配置实际的后端 API 地址
3. 微信小程序需要配置服务器域名白名单
4. 如果使用微信登录，需要配置微信开放平台相关信息
