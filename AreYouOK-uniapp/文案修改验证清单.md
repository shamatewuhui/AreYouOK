### Q1: 修改后看不到变化?

**A**: 清除缓存并重新编译
```bash

工具 → 清除缓存 → 全部清除

# 2. 重新编译
npm run build
```

### Q2: 按钮文字显示不完整?

**A**: 检查CSS样式,确保字号和间距合适
```scss
.button-text {
  font-size: 56rpx;
  letter-spacing: 6rpx;
}
```

**A**: 检查pages.json配置是否正确
