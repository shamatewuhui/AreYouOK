# 本地编译使用说明

## ✅ 编译已完成！

项目已成功编译为微信小程序格式，编译输出目录：`dist/dev/mp-weixin`

## 📱 在微信开发者工具中打开

1. **打开微信开发者工具**
2. **选择"导入项目"**
3. **项目目录**：选择 `AreYouOK-uniapp/dist/dev/mp-weixin`
4. **AppID**：`wx9d3afb740a541e4e`
5. 点击"导入"

## 🔄 重新编译

如果修改了源码，需要重新编译：

```bash
cd AreYouOK-uniapp
node build.js
```

然后刷新微信开发者工具中的项目。

## ⚠️ 注意事项

1. **API 地址配置**：需要在 `dist/dev/mp-weixin/utils/api.js` 中修改 `BASE_URL` 为实际的后端 API 地址

2. **小程序配置**：确保在微信公众平台配置了服务器域名白名单

3. **文件修改**：如果直接修改编译后的文件，重新编译会覆盖你的修改

## 📁 项目结构

```
AreYouOK-uniapp/
├── pages/              # 源码页面（Vue文件）
├── utils/              # 工具函数（源码）
├── App.vue            # 应用入口
├── pages.json         # 页面配置
├── manifest.json      # 应用配置
├── build.js           # 编译脚本
└── dist/              # 编译输出目录
    └── dev/
        └── mp-weixin/ # 微信小程序代码（在微信开发者工具中打开此目录）
            ├── app.js
            ├── app.json
            ├── app.wxss
            ├── pages/
            │   └── index/
            │       └── index/
            │           ├── index.js
            │           ├── index.wxml
            │           └── index.wxss
            └── utils/
```

## 🐛 如果遇到问题

1. **检查编译输出**：确保 `dist/dev/mp-weixin` 目录存在且包含必要文件
2. **检查 app.json**：确保页面路径正确
3. **查看控制台**：在微信开发者工具中查看控制台错误信息
4. **重新编译**：删除 `dist` 目录后重新运行 `node build.js`
